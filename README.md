# dotfiles
## Steps: 
1. Install [NeoVim](https://neovim.io/)
2. Delete any local font installs and download desired terminal font from [here](https://github.com/ryanoasis/nerd-fonts#patched-fonts)
3. Make an `init.vim` in /home/cooper/.config/nvim/
4. Copy contents of `init.vim`
5. Install [VimPlug](https://github.com/junegunn/vim-plug)
6. Run `:PlugInstall` and start praying things work
